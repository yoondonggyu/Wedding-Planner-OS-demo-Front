# Wedding Planner OS - React Version

Vue.js 프로젝트를 React로 변환한 버전입니다.

## 설치 및 실행

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 빌드
pnpm build
```

## 변환 상태

- ✅ 프로젝트 구조 및 설정 파일
- ✅ Router 설정 (React Router)
- ✅ Auth Store (Zustand)
- ✅ API Hook
- ✅ App 컴포넌트 기본 구조
- ⏳ View 컴포넌트들 (진행 중)
- ⏳ 공통 컴포넌트들 (진행 중)

## 주요 변경사항

- Vue 3 → React 18
- Vue Router → React Router v6
- Pinia → Zustand
- Composition API → React Hooks
- Vue SFC → JSX/TSX
